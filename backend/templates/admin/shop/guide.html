后台使用指南

你的站点只卖 Codex 卡密，日常补货直接使用“一键导入”即可。

1. 一键导入 Codex 卡密

填写价格和说明，粘贴卡密或上传 TXT/CSV，系统会自动创建商品并补充库存。

一键导入

2. 查看商品

导入后会出现唯一商品“Codex 卡密”，可以查看库存、价格和前台上架状态。

商品列表

3. 前台购买测试

刷新前台首页，看到 Codex 卡密库存增加后，就可以下单并模拟支付测试发货。

打开前台

4. 查看订单

用户下单和支付后的记录会出现在订单列表，已支付订单会显示发货内容。

查看订单

5. 配置公告

公告会展示在前台首页，可用于说明购买须知、售后规则或维护通知。

新增公告

6. 修改站点信息

站点名称、售后联系方式等内容在这里配置，不需要改代码。

站点配置
最常用流程： Codex 卡密一键导入 → 前台刷新 → 下单测试。
不建议直接在“卡密库存”里新增，因为卡密需要经过系统加密；请使用“一键导入”入口。
